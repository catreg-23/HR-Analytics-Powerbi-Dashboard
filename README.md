HR-Analytics-Powerbi-Dashboard
An interactive HR Analytics Dashboard built using Power BI to analyze employee attrition, workforce demographics, job satisfaction, salary distribution, and key HR metrics through dynamic visualizations.
An interactive HR Analytics Dashboard built using Power BI to support data-driven HR decision-making. The project focuses on employee attrition, workforce demographics, job satisfaction, salary analysis, departmental performance, and key HR KPIs. It incorporates data cleaning, transformation, modeling, and DAX calculations to create dynamic dashboards with drill-through reports, slicers, and interactive visualizations.
